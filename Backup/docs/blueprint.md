# **App Name**: FinTrack

## Core Features:

- Data Visualization: Visualize income and expenses through interactive charts and graphs.
- Categorization: Classify all financial entries into predefined categories.
- Data Export: Allow export to spreadsheets

## Style Guidelines:

- Primary color: Dark purple for trustworthiness.
- Neutral background: Black for content contrast.
- Accent color: Navy blue for positive transactions and calls to action.
- Secondary color: White for text and other elements.
- Clean and readable sans-serif fonts.
- Use minimalist icons to represent transaction categories.
- Dashboard layout with clear sections for income, expenses, and charts.
- Smooth transitions and animations for a user-friendly experience.
